##已完成功能

+ 启动动画
+ 主题日报的获取
+ 今日热点新闻的轮播
+ 使用谷歌官方的`swpieRefreshLayout`实现下拉刷新
+ 使用`RecyclerView`代替`ListView`
+ 标记已读文章
+ `viewPager`的自定义
+ `RecyclerView`实现底部自动加载更多
+ **努力的**使用了MVP结构
+ `CollapsingToolbarLayout` + `AppBarLayout`实现滚动视差特效
+ 尽量的符合`Material Design`规范


##依赖的第三方库

+ [android-async-http][2]
+ [Gson][3]
+ [ButterKnife][4]
